3D Model Files

Place your 3D model files (.glb) in this directory.

Required files:
- table_wood.glb - A 3D model of a wooden table
- Meuble ID-1.glb - A 3D furniture model

You can download sample GLB files from:
- https://modelviewer.dev/examples/assets/
- https://sketchfab.com/ (download as GLB format)
- https://poly.pizza/ (free 3D models)

Note: Ensure that the models are optimized for web and mobile platforms. 
Ideally, keep file sizes under 10MB for better performance.
